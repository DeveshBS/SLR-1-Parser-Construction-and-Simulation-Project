# SLR(1) Parser Construction & Simulation in C++

Complete SLR(1) parser generator that builds FIRST/FOLLOW sets, LR(0) items, canonical collection, ACTION/GOTO tables, and an interactive shift‑reduce simulator.

## Features

- FIRST & FOLLOW set computation for any context‑free grammar
- LR(0) item closure and GOTO functions
- Canonical collection of states (DFA)
- Automatic ACTION and GOTO table construction
- Interactive parsing simulation with step‑wise stack tracing
- Supports augmented grammar, ε‑productions, and conflict detection
